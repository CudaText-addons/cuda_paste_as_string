Paste as String
===============

It helps to insert some code as a string

----

Default settings for languages:

+ C#
+ C++
+ Java
+ JavaScript
+ Perl
+ PHP
+ Ruby
+ VBS

And it is very easy to setup others.

Usage
-----

For example, the text in the clipboard:
```sql
SELECT field1
FROM "table"
WHERE field2 = 'string'
```
